# Trip Navigator

A desktop-based travel management system built using **Java Swing**, **AWT**, and **MySQL**.  
The application helps users plan and manage trips, store destination details, and view itineraries through an interactive GUI.

# Features
- User-friendly GUI built with Swing and AWT  
- Add, view, update, and delete trip details  
- Connects to MySQL database for data storage  
- Displays travel history and cost calculations  
- Login and registration system for users  

# Technologies Used
- **Java (Core + Advanced)**
- **Swing & AWT** (for GUI)
- **MySQL** (for database)
- **JDBC** (for database connectivity)
- **Eclipse**
